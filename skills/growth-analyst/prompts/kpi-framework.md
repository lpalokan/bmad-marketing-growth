# Kpi Framework

<instructions>Define a KPI framework matched to the SaaS stage</instructions>
<process>
1. Ask about the current stage (Pre-PMF, PMF, Scale) and the business model
2. Identify the 3-5 North Star metrics for that stage
3. Define the secondary KPIs that feed the North Star
4. Build the hierarchy: North Star -> KPIs -> Metrics -> Inputs
5. Recommend fitting tracking tools
6. Set review cadences (daily, weekly, monthly)
</process>
<output_format>KPI framework with hierarchy, definitions, and review cadence</output_format>
