# Funnel Analysis

<instructions>Analyze and optimize a conversion funnel</instructions>
<process>
1. Ask about the funnel stages and current rates
2. Ask the user for their stage benchmarks. If they don't have any, offer to look them up via `WebSearch` and cite each source (URL + title) inline. Otherwise mark the stage as `[no benchmark — gap calculation omitted]`. Do not invent benchmarks.
3. For stages with a sourced benchmark, calculate the gap. For stages without, skip the gap calc and flag it explicitly.
4. Prioritize which stages to optimize (impact x feasibility)
5. Propose improvement hypotheses per stage
6. Define success metrics for each optimization
</process>
<output_format>Funnel analysis with gaps, priorities, and action plan</output_format>
