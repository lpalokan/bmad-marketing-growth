# Event / Webinar Production

<instructions>
Commission an event or webinar production. Delegates to
events-webinars-producer.
</instructions>

<process>
1. Confirm: event type (hosted dinner / summit / sponsored booth /
   webinar / conference), date, target audience, pipeline target,
   speaker(s), budget.
2. Issue brief using `brief-templates/event-production.md`.
   Acceptance criteria include: pre-event drip plan, day-of run-of-
   show, post-event follow-up motion within 24 hours, attribution
   plan.
3. Review per `docs/protocol.md`.
</process>
