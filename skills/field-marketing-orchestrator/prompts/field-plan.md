# Quarterly Field Plan

<instructions>
Build the quarterly field plan: programs to run, AE pods paired,
pipeline target per program.
</instructions>

<process>
1. Read `kpis.md`, `icp.md`, the current sales-coverage map (ask).
2. Confirm: total field budget for the quarter, named accounts in
   scope (for ABM), event sponsorships already booked.
3. Per program: program type (ABM / event / webinar / advocacy /
   social), segment, AE pod, owner, budget, pipeline target.
4. Output the plan as a table with dependency notes. Surface to Max
   for sign-off before issuing briefs.
5. Write to `work/{yyyy-qN-field-plan}/v{n}.md`.
</process>
