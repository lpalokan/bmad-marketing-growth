# Customer Advocacy Program

<instructions>
Commission a customer-advocacy or reference program. Delegates to
customer-advocacy-references.
</instructions>

<process>
1. Confirm: program type (case study / reference call program /
   advisory board / advocate community), target advocate count,
   incentive structure, CSM partnership.
2. Issue brief using `brief-templates/customer-advocacy.md`.
   Acceptance criteria include: advocate identification process,
   permission flow, advocate ROI tracking, evergreen reference health
   check.
3. Review per `docs/protocol.md`.
</process>
