# ABM Program

<instructions>
Commission an ABM program. Delegates to abm-strategist.
</instructions>

<process>
1. Confirm: named-account list, segment, paired AE pod, sales-stage
   focus (top-of-funnel awareness vs deal-acceleration), budget,
   measurement plan.
2. Issue brief using `brief-templates/abm-program.md`. Acceptance
   criteria include: account plan per tier, multi-channel orchestration,
   measurement that doesn't double-count.
3. Review per `docs/protocol.md`.
</process>
