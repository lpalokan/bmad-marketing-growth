# Top-10 FAQ

<instructions>
Anticipate the top 10 objections / questions and answer each.
</instructions>

<process>
1. Read the launch narrative and any in-scope competitive teardowns.
2. Pull the top objections from the teardown's "How we sell against
   them" + 3-5 product-specific anticipated questions.
3. For each Q, write a 1-3 sentence A. Cite sources for any number.
4. Tag each Q with audience: AE / SE / BDR / CSM / customer.
5. Output as `work/{brief_id}/v{n}-faq.md`.
</process>
