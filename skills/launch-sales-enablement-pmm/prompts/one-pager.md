# One-Pager

<instructions>
Produce a 1-page customer-facing one-pager.
</instructions>

<process>
1. Use the approved launch narrative as input. If not approved yet,
   refuse and ask the orchestrator to approve LN first.
2. Structure:
   - Headline (≤8 words; matches one-line story).
   - Subhead (one sentence; matches 30-second elevator).
   - Why now (one paragraph).
   - What you get (3 bullets, each ≤14 words).
   - Proof (1-2 customer quotes or stats with sources).
   - CTA (single primary CTA per persona).
3. Strip jargon. Run a final pass against `brand-voice.md` don't list.
4. Output as `work/{brief_id}/v{n}-onepager.md`.
</process>
