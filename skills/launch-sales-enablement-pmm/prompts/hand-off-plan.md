# Down-stream Hand-off Plan

<instructions>
Plan the briefs that go to PR Comms, Digital, Field, and Content for a
launch. Output is a hand-off plan; the actual briefs are written by the
respective orchestrators after their reviews.
</instructions>

<process>
1. Read the approved launch narrative.
2. For each downstream owner — PR Comms, Digital, Field, Content —
   write a one-paragraph hand-off note: what the launch needs from
   them, the brief-template path they should use, the dependency dates.
3. Include a single timeline view: T-6 weeks through T+2 weeks, with
   owner per row.
4. Output as `work/{brief_id}/v{n}-handoff.md`.
</process>
