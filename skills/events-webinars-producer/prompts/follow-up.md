# 24-Hour Follow-Up Motion

<instructions>
Build the post-event follow-up motion that fires within 24 hours.
</instructions>

<process>
1. Segment follow-up:
   - Attended + engaged: AE direct outreach within 24 hours.
   - Attended + passive: nurture handoff to lifecycle-activation.
   - Registered + no-show: recording + on-demand CTA.
2. For each segment: message copy, channel (email + LinkedIn from
   AE if applicable), CTA.
3. Automation: which MAP workflow handles the segmentation; hand off
   to marketing-automation-engineer if it doesn't exist.
4. AE briefing: 1-page summary per attended account with key
   moments / questions raised.
5. Write into `v{n}.md`.
</process>
