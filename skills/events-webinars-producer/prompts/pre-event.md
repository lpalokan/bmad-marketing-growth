# Pre-Event Drip Plan

<instructions>
Build the pre-event drip plan.
</instructions>

<process>
1. Read brief, registrant / invitee list profile (ask).
2. Drip plan: at least 3 touches before the event.
   - T-14: save the date / why you should care.
   - T-7: agenda + speaker spotlight.
   - T-1: tomorrow reminder with calendar attach.
3. Segment: registered vs invited; tailor copy per segment.
4. Channel per touch: email primary; SMS for T-1 if available and
   consented.
5. Show-up forecast: cite a baseline source (prior events or user-
   supplied).
6. Write into `v{n}.md`.
</process>
