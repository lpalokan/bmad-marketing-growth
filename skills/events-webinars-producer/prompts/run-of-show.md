# Day-of Run-of-Show

<instructions>
Build the day-of run-of-show with timing, owner, contingency.
</instructions>

<process>
1. Walk the event from check-in (or webinar-live) through close-out
   in 15-minute slots.
2. Per slot: time, item, owner, AV needs, contingency.
3. Speaker prep: confirmation call (T-2 day) + tech check (T-3 hours).
4. Hosts / MC script outline.
5. Engagement asks: poll questions, live Q&A flow, post-it/whiteboard
   prompts for in-person.
6. Write into `v{n}.md`.
</process>
