# Quarterly Digital Plan

<instructions>
Build the quarterly digital plan: channel allocation, targets per
channel, and the specialist briefs that will execute it.
</instructions>

<process>
1. Read `kpis.md`, `positioning.md`, `icp.md`, and the latest growth
   audit if available.
2. Confirm with the user: total budget for the quarter, pipeline
   target, target MQL→SQL conversion baseline (ask, do not invent).
3. Allocate budget per channel (paid search / paid social-demand /
   web/CRO / automation / SEO infra). Cap allocation at the user's
   confirmed budget. Each channel allocation includes: target
   pipeline contribution, target CPL or CPA, and the lead specialist.
4. Output the plan as `work/{yyyy-qN-digital-plan}/v{n}.md` with the
   table, the dependency notes, and the list of follow-on briefs to
   issue.
5. Surface the plan to Max (CMO) for cross-domain sign-off before
   issuing the specialist briefs.
</process>
