# Web / CRO Sprint

<instructions>
Commission a website or CRO sprint. Delegates to web-cro-specialist.
</instructions>

<process>
1. Confirm scope with the user: which page(s), what hypothesis is being
   tested, what change is being made, what success looks like.
2. Issue a brief using `brief-templates/web-cro.md`. Acceptance
   criteria include: hypothesis stated, sample-size calculation
   sourced, measurement plan, rollback plan, accessibility check.
3. Review per `docs/protocol.md`.
</process>
