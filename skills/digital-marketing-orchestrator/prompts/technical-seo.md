# Technical SEO Sprint

<instructions>
Commission a technical-SEO sprint: infra, schema, Core Web Vitals,
hreflang, indexation. Delegates to technical-seo-engineer.
</instructions>

<process>
1. Confirm scope: site / subdirectory in scope, audit triggered by
   what (traffic drop, CWV failure, hreflang setup for new locale,
   schema markup for a content type).
2. Issue brief using `brief-templates/technical-seo.md`. Acceptance
   criteria: audit findings table, fix-priority matrix (impact x
   effort), implementation tickets, post-fix verification plan.
3. Review per `docs/protocol.md`.
</process>
