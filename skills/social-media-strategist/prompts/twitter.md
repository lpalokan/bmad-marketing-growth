# Twitter / X Content

<instructions>
Write Twitter / X content directly: threads, single posts, ghost-
written executive posts.
</instructions>

<process>
1. Confirm: format (single tweet / thread / quote-post), author voice
   (brand voice from `brand-voice.md` or speaker voice from a per-
   speaker `voice-notes.md`), business outcome (awareness / engagement
   / link clicks).
2. For threads: hook tweet (≤ 280 chars, earns scroll-stop), 5-15
   body tweets (each ≤ 280 chars, each readable standalone), close
   tweet (CTA / synthesis).
3. Style: short sentences, active voice, no engagement-bait. Cite
   sources for any number.
4. Verify against `brand-voice.md` don't list.
5. Output: ready to schedule (number tweets 1/N, 2/N, etc.).
</process>
