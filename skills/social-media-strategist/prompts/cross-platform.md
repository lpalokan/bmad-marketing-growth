# Cross Platform

<instructions>
Create coordinated cross-platform strategy with editorial calendar.
Transform one core content piece into platform-specific formats.
</instructions>
<process>
1. Define core content piece (blog, video, announcement)
2. Map to platform adaptations:
   - Twitter: Thread format
   - LinkedIn: Professional post
   - YouTube: Short or long-form
   - Reddit: Discussion starter
   - Discord: Community announcement
   - Instagram: Carousel or Reel
   - TikTok: Short-form video
   - Pinterest: Visual pin with SEO
3. Create editorial calendar with timing
4. Output: Content matrix + publication schedule
</process>
