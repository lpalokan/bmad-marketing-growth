# Per-Persona Value Props

<instructions>
Produce a value-prop set per ICP persona. One line per persona;
maximum five personas.
</instructions>

<process>
1. Load `icp.md`. Extract the named personas (titles + segments).
2. For each persona, write a value prop in this shape:
   "{Persona} can {outcome} {within timeframe} by {mechanism}, instead
   of {current alternative}."
3. Cite the source for any quantified outcome.
4. If a persona's outcome can't be sourced, write `[outcome: needs
   source]` instead of guessing.
5. Write into `work/{brief_id}/v{n}.md`.
</process>
