# Messaging Pillars

<instructions>
Produce three messaging pillars with proof points, mapped to the value
pyramid. Output is part of `v{n}.md`.
</instructions>

<process>
1. Read brief, `positioning.md`, `icp.md`, `brand-voice.md`.
2. Identify the three value-pyramid layers in `positioning.md`. Each
   pillar maps to exactly one layer.
3. For each pillar, write: pillar name (≤4 words), one-sentence
   articulation in brand voice, 3+ proof points each with an inline
   source (URL, internal doc path, or `user-provided: <date>`).
4. Anti-positioning: list at least two claims this pillar does NOT
   make (to prevent scope creep).
5. Verify against `brand-voice.md` do/don't lists. If any phrase
   appears on the don't list, replace it.
6. Write into `work/{brief_id}/v{n}.md`.
</process>
