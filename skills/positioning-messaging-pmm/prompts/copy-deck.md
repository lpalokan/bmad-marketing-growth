# Channel Copy Deck

<instructions>
Adapt the approved messaging to a specific channel's constraints.
Output is per-channel rows.
</instructions>

<process>
1. Confirm channel(s) in scope from the brief: web hero, web subhead,
   paid headline, paid description, cold email subject, cold email
   body, sales-call opener, etc.
2. For each row, write character-counted copy in brand voice. Include
   the character/word limit explicitly.
3. Where a row needs a number, pull it from the proof points in the
   approved messaging pillars; cite inline. If absent, use `[number:
   needs source]`.
4. Label any sample numbers `Example — illustrative, not benchmarks.`
5. Write the deck as a table into `work/{brief_id}/v{n}.md`.
</process>
