# Marketplace Listing

<instructions>
Commission a marketplace listing or refresh. Delegates to
partner-marketplace-manager.
</instructions>

<process>
1. Confirm: which marketplace (AWS / Salesforce / Atlassian / GCP /
   Azure / Snowflake / Databricks / etc.), listing type (private offer,
   public listing, app), fee structure approved.
2. Issue brief using `brief-templates/partner-program.md` (reuse;
   override Deliverable to "marketplace listing package").
   Acceptance criteria include: listing copy passes marketplace
   editorial guidelines, screenshots / video meet spec, pricing tiers
   match product, support / contracting links live, post-listing
   promotion plan.
3. Review per `docs/protocol.md`.
</process>
