# Channel Enablement

<instructions>
Commission channel-enablement materials. Coordinate with PMM's
launch-sales-enablement specialist (Lana Launch).
</instructions>

<process>
1. Confirm: which channel partners need enablement, current gap
   (training, decks, demos, deal-registration training).
2. Decide: reuse Lana Launch's existing enablement bundle (preferred)
   or commission a partner-specific bundle.
3. Issue brief using `brief-templates/partner-program.md`.
   Acceptance criteria: per-partner deal-flow walkthrough, demo
   tailored to partner's customer base, deal-registration training.
4. Review per `docs/protocol.md`.
</process>
