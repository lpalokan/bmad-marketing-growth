# Media Relations Push

<instructions>
Commission a media-relations push. Delegates to media-relations-specialist.
</instructions>

<process>
1. Confirm: anchor news, embargo window, exclusive vs broad, target
   outlets, target byline writers.
2. Issue brief using `brief-templates/media-relations.md`. Acceptance
   criteria: pitch list with reasoning per outlet, embargo plan,
   press release ready for legal, post-publication monitoring.
3. Review per `docs/protocol.md`.
</process>
