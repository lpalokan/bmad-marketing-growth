# Executive Communications

<instructions>
Prep an executive comms moment (keynote, board update, all-hands
external, podcast).
</instructions>

<process>
1. Confirm: speaker, audience, format, time, takeaway.
2. Structure: one-line story → 3 main points → proof per point →
   call to action / call to belief.
3. Voice: speaker's voice (not brand voice — speakers have their own
   cadence). Capture in a per-speaker `voice-notes.md` sidecar.
4. Q&A prep: top 10 likely questions.
5. Write to `work/{yyyy-mm-exec-{topic}}/v{n}.md`.
</process>
