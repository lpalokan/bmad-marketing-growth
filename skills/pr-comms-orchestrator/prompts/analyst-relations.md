# Analyst Relations Engagement

<instructions>
Commission an analyst-relations engagement. Delegates to
analyst-relations-specialist.
</instructions>

<process>
1. Confirm scope: routine briefing, MQ / Wave submission prep,
   inquiry, post-publication response.
2. Issue brief using `brief-templates/analyst-relations.md`. Acceptance
   criteria: prep deck mapped to analyst's evaluation criteria,
   spokesperson per topic, expected outcome.
3. Review per `docs/protocol.md`.
</process>
