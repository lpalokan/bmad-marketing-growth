# Quarterly PR Plan

<instructions>
Build the quarterly PR plan: anchor moments, analyst cadence, exec
comms calendar.
</instructions>

<process>
1. Anchor moments: 1-3 launches or proof points per quarter (sync
   with PMM `launch-narrative`).
2. Analyst cadence: which analysts to brief, which submissions are
   due (MQ / Wave / SoftwareReviews / etc. — confirm with user).
3. Executive comms: who's keynoting where; preparation cadence.
4. Press distribution: where (which outlets / vehicles), when.
5. Write to `work/{yyyy-qN-pr-plan}/v{n}.md`.
</process>
