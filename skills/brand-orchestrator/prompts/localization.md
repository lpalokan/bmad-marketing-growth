# Localization Rollout

<instructions>
Commission a localization rollout. Delegates to
localization-international-lead.
</instructions>

<process>
1. Confirm: target locales, in-scope assets (web, paid, email, sales
   collateral), launch date, budget, owner per locale.
2. Issue brief using `brief-templates/localization.md`. Acceptance
   criteria include: per-locale voice adaptation guide, glossary,
   translation memory setup, QA plan, in-market reviewer per locale.
3. Review per `docs/protocol.md`.
</process>
