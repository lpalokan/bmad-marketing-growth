# Marketplace Listing Package

<instructions>
Build a marketplace listing package for AWS / Salesforce / Atlassian /
GCP / Azure / Snowflake / Databricks / etc.
</instructions>

<process>
1. Identify the marketplace and pull its editorial guidelines URL.
2. Listing copy: short description + long description per guidelines;
   pricing tiers; SLAs; support flow.
3. Visual assets: logo, hero, screenshots, video (spec varies by
   marketplace — confirm).
4. Pricing structure: SaaS contract / private offer / metered;
   align with our product packaging.
5. Editorial pre-check: walk through the guidelines line by line.
6. Post-listing promotion plan: which channels announce; how the
   marketplace customer journey is wired to our CRM.
7. Write into `v{n}.md`.
</process>
