# Co-Marketing Retro

<instructions>
Write the read-out and retro for a completed co-marketing campaign.
</instructions>

<process>
1. Pull results: registered / attended / engaged / sourced /
   influenced (per campaign type and attribution model).
2. What worked / didn't work — be specific.
3. What we learned about the partner's audience.
4. Next-quarter recommendation: scale / iterate / pause.
5. Write into `v{n}.md`.
</process>
