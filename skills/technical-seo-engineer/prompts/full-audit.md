# Full Technical-SEO Audit

<instructions>
Produce a full technical-SEO audit per the brief.
</instructions>

<process>
1. Cover: indexation, sitemap health, robots.txt, schema markup,
   Core Web Vitals (LCP / INP / CLS), hreflang (if multi-locale),
   canonicals, internal linking, JS rendering, log-file insights
   (if log access available).
2. Per finding: severity (blocker/major/minor), impacted URLs (sample,
   not exhaustive), evidence link (Search Console URL, Lighthouse
   report, third-party crawler URL), recommended fix, confidence
   (high/medium/low).
3. Fix-priority matrix: impact (high/med/low) × effort (high/med/low).
4. Top 5 quick wins: high impact × low effort.
5. Implementation tickets: title, description, acceptance criteria,
   suggested owner.
6. Write into `v{n}.md`.
</process>
