# Indexation Troubleshooting

<instructions>
Diagnose indexation issues against Search Console signals.
</instructions>

<process>
1. Ask for: Search Console URL inspection screenshots for affected
   URLs, coverage report categories with counts.
2. Categorise the issues:
   - Crawled - currently not indexed.
   - Discovered - currently not indexed.
   - Excluded by noindex / robots.
   - Duplicate without user-selected canonical.
   - Soft 404.
3. Per category: root-cause hypothesis, evidence required to confirm,
   fix list.
4. Distinguish site-side fixes from Google-side delays (cite source
   for any reported Google issue).
5. Write into `v{n}.md`.
</process>
