# Commission Content-SEO Brief

<instructions>
Commission a content-SEO piece of work. Delegates to
content-seo-strategist (Quinn).
</instructions>

<process>
1. Confirm scope: what (keyword research, content-gap analysis, topic
   cluster), target topic / category, deadline, intended downstream
   use (article brief, pillar page, distribution plan).
2. Issue brief using `brief-templates/content-seo.md`. Acceptance
   criteria include: search intent classification, SERP analysis
   summary, target page type per keyword cluster, content-gap matrix,
   priority recommendation with rationale.
3. Review per `docs/protocol.md`.
</process>
