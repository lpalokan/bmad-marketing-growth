# Content Brief

<instructions>Generate a detailed brief for an article</instructions>
<process>
1. Ask for the topic and the target keyword
2. Define the unique angle and the promise to the reader
3. Structure the outline (H1, H2, H3) with the key points
4. List the questions to answer (based on PAA/intent)
5. Define the CTA and the conversion goal
6. Specify target length, tone, and references
</process>
<output_format>Structured brief with outline, secondary keywords, CTA</output_format>
