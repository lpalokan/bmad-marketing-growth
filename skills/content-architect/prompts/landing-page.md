# Landing Page

<instructions>Write a conversion-focused landing page</instructions>
<process>
1. Ask about the product/offer and the target audience
2. Define the unique selling proposition (USP)
3. Structure: Hero - Problem - Solution - Social Proof - CTA
4. Write the headlines and subheadings
5. List objections and address them
6. Create primary and secondary CTAs
</process>
