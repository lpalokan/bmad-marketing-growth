# Funnel Scorecard

<instructions>
Build (or refresh) the funnel scorecard the team uses weekly.
</instructions>

<process>
1. Read `kpis.md`. Pull the 5-10 KPIs that map to the funnel.
2. Per row:
   - Stage / metric name.
   - Definition.
   - Current value + source URL.
   - Target value + horizon.
   - Owner.
   - Trend (last 6 weeks if available).
3. Highlight any metric without a source — those need fixing in
   `kpis.md`.
4. Output the scorecard table and a one-paragraph narrative summary.
5. Write into `v{n}.md`.
</process>
