# Funnel Audit

<instructions>
Commission a funnel audit. Delegates to experimentation-funnel-lead.
</instructions>

<process>
1. Confirm: funnel stages in scope, time window, segmentation
   dimensions, the question the audit must answer.
2. Issue brief using `brief-templates/experiment.md` (reuse — the
   acceptance criteria are similar; add an "Audit deliverable"
   section in the brief overriding Deliverable).
3. Pair with Pixel Metrics (growth-analyst) for measurement support.
4. Review per `docs/protocol.md`.
</process>
