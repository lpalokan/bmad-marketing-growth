# Experiment

<instructions>
Commission a single experiment. Delegates to experimentation-funnel-lead.
</instructions>

<process>
1. Confirm: hypothesis, primary metric, baseline value (ask — do
   not invent), expected lift, sample-size availability.
2. Issue brief using `brief-templates/experiment.md`. Acceptance
   criteria include: testable hypothesis, sample-size calculation,
   primary + 2 guardrail metrics, randomization unit, rollback plan,
   read-out plan.
3. Review per `docs/protocol.md`. On APPROVED, decide rollout vs
   discard with the user before freezing.
</process>
