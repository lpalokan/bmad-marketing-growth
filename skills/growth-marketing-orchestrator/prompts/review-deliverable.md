# Review Deliverable

<instructions>
Review a specialist's `v{n}.md` against its `brief.md` per `docs/protocol.md`.
</instructions>

<process>
1. Read `brief.md`, latest `v{n}.md`, `state.yaml`.
2. Score against the brief's Acceptance Criteria item by item.
3. Verdict: APPROVED / NEEDS_REVISION / ESCALATED.
4. Write `v{n}-review.md` per schema; update `state.yaml`.
</process>
