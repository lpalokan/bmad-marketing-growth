# Lifecycle / Activation Program

<instructions>
Commission a lifecycle or activation program. Delegates to
lifecycle-activation specialist.
</instructions>

<process>
1. Confirm: program objective (activation, nurture, expansion,
   win-back), target segment, channels in scope (email / in-app /
   push / SMS), expected sequence length, gating asset (if any).
2. Issue brief using `brief-templates/lifecycle-program.md`.
   Acceptance criteria include: cross-channel sequence map, message-
   level copy per step, segmentation rules, exit criteria,
   measurement plan, suppression rules.
3. Review per `docs/protocol.md`.
</process>
