# Pitch List

<instructions>
Build a pitch list with rationale per outlet and per writer.
</instructions>

<process>
1. Read brief, the launch narrative, the anchor news.
2. Per outlet: outlet name, byline writer, recent coverage that shows
   beat fit (cite URL + date), why this news matters to them
   specifically, preferred contact mode.
3. Tier: T1 (exclusive candidates), T2 (under embargo broad), T3
   (post-embargo broad).
4. Verify: every writer's latest beat — outlets change writers
   constantly; never pitch from a stale list.
5. Write into `v{n}.md`.
</process>
