# Journalist Outreach Sequence

<instructions>
Build the outreach sequence per tier.
</instructions>

<process>
1. T1 (exclusive): personalised email to one writer; embargo terms
   explicit; ≤ 150 words; one CTA (interview slot).
2. T2 (broad embargo): personalised opener + concise pitch + embargo
   details; ≤ 200 words.
3. T3 (post-embargo broad): wire + targeted follow-up to T3 writers.
4. Follow-up cadence: T+24h soft reminder; T+72h close.
5. Calendar offer: link to scheduling tool with prepped slots.
6. Write into `v{n}.md`.
</process>
