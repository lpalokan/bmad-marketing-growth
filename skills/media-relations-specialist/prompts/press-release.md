# Press Release

<instructions>
Draft a press release ready for legal review.
</instructions>

<process>
1. Structure: headline → subhead → dateline → lede (most important
   in first sentence) → body (3-5 paragraphs, inverted pyramid) →
   quotes (2-3, attributed) → boilerplate → media contact.
2. Quotes: attributed; voice = speaker's voice (not brand voice).
3. Boilerplate: pull latest legal version (ask, do not invent).
4. Verify no benchmark is asserted without a source.
5. Write into `v{n}.md`.
</process>
