# Post-Publication Monitoring

<instructions>
Plan post-publication coverage monitoring.
</instructions>

<process>
1. Tool: ask which monitoring tool is in use (Meltwater, Cision,
   Muck Rack, etc.). Configure search terms.
2. Coverage tracking: outlet, writer, sentiment, audience reach
   (cite source for reach).
3. Response plan: corrections within 24h; positive coverage
   amplified by Field / Social via Nova.
4. Report cadence: end-of-week summary.
5. Write into `v{n}.md`.
</process>
