# Locale QA Plan

<instructions>
Build the QA plan for a locale launch.
</instructions>

<process>
1. Three QA layers:
   - Linguistic review (translator output vs glossary, TM, voice
     guide).
   - In-market review (named reviewer in country reads with buyer
     persona in mind).
   - Technical review (encoding, layout overflow, date / number /
     currency formatting, hreflang — coordinate with Tek Crawl).
2. Sign-off table: layer × reviewer × deadline × pass-criteria.
3. Pre-launch checklist: dry run of the first 5 assets in each
   channel.
4. Write into `v{n}.md`.
</process>
