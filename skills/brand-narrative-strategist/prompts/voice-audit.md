# Voice Audit

<instructions>
Audit a set of existing assets against `brand-voice.md`.
</instructions>

<process>
1. Ask user for sample assets (URLs or copy blocks).
2. Per asset: score against voice attributes (each pass / fail / mixed)
   and flag every do/don't-list violation with the exact quote.
3. Pattern-recognise: which channel / team / vendor is the systemic
   miss.
4. Recommendation: 3-5 specific edits per asset; 2-3 systemic process
   fixes.
5. Write into `v{n}.md`.
</process>
