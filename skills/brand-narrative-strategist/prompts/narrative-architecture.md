# Narrative Architecture

<instructions>
Build the narrative architecture: through-line + chapters by funnel
stage + exemplar per chapter.
</instructions>

<process>
1. Read brief, `positioning.md`, `brand-voice.md`, `icp.md`.
2. Distill the through-line — one sentence. If you need >25 words, you
   don't have it yet. Iterate.
3. Chapters by funnel stage:
   - Awareness: hook + insight + outlook.
   - Consideration: differentiation + proof.
   - Decision: clarity + risk reduction.
   - Expansion: depth + community.
4. Per chapter: one exemplar asset in brand voice (a headline + 30-word
   body). Mark anything fabricated and ask for the source.
5. Localization-readiness notes per chapter: which phrases are idiomatic
   and will not survive translation.
6. Write into `v{n}.md`.
</process>
