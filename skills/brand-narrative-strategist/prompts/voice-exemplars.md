# Voice Exemplars

<instructions>
Produce voice exemplars across the channels the team uses.
</instructions>

<process>
1. List channels in scope from the brief: web hero, paid headline,
   cold email, sales-call opener, support reply, PR opener, etc.
2. Per channel, write 3 exemplars in brand voice with explicit
   character counts.
3. For each exemplar, annotate which voice attribute(s) it demonstrates
   and which don't-list word it carefully avoids.
4. Write into `v{n}.md`.
</process>
