# Post-Publication Response

<instructions>
Plan the response to a published analyst report.
</instructions>

<process>
1. Read the report. Note: rating, strengths cited, cautions cited,
   competitive positioning.
2. Internal comms: brief CEO / CFO / CRO / product leadership;
   distinguish "fix this" cautions from "explain this" cautions.
3. External response: blog post if positive movement (PMM owns the
   narrative); silent if neutral; thoughtful response if cautions
   warrant.
4. Analyst follow-up: thank-you + 1-2 substantive responses to
   cautions. Do NOT debate.
5. Customer comms: case study program for any cautions that contradict
   our customer evidence.
6. Write into `v{n}.md`.
</process>
