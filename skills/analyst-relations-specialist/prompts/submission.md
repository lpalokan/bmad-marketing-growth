# MQ / Wave / MarketScape Submission Prep

<instructions>
Prep a major analyst submission.
</instructions>

<process>
1. Read the analyst's published evaluation criteria for THIS year
   (changes annually).
2. Map each criterion to evidence: product feature + roadmap +
   customer evidence + market presence.
3. For each criterion, score honestly: where we're strong, where
   we're weak, what's improving.
4. Reference customers: select 5-10 named (with permission) who can
   speak to specific criteria; brief them.
5. Submission package: deck, doc responses, supporting links. Legal
   review before send.
6. Internal expectation-setting: rating movement realistic, not
   wishful.
7. Write into `v{n}.md`.
</process>
