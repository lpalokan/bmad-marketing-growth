# Quarterly Competitive Landscape

<instructions>
Refresh the competitive landscape view: who's in the category, where
they sit on the positioning map, what's moved this quarter.
</instructions>

<process>
1. Ask the user for the category boundaries (which competitors count,
   which adjacent categories to mention but exclude).
2. For each in-scope competitor: one-paragraph status update covering
   any of the following that changed this quarter — funding, M&A,
   pricing, leadership, major product launch, analyst rating.
3. Positioning map: place each competitor on a 2x2 against axes
   relevant to our ICP (ask the user for axis choice — do not pick
   for them).
4. Net call-out: 3 things our PMM org should react to this quarter.
5. Write `v{n}.md`.
</process>
