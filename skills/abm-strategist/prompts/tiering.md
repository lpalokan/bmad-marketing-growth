# Account Tiering

<instructions>
Tier the named-account list into 1:1 / 1:few / 1:many.
</instructions>

<process>
1. Read brief and named-account list.
2. Tier criteria:
   - 1:1: top 10-20 accounts; bespoke touches, named buyer mapping,
     individual content.
   - 1:few: 30-80 accounts grouped by segment / industry; semi-
     bespoke (cluster content).
   - 1:many: 200+ accounts; programmatic, audience-segment level.
3. Per tier: rationale, bandwidth required, expected touch density,
   AE pairing model.
4. Output the tiering table with named accounts per tier.
5. Write into `v{n}.md`.
</process>
