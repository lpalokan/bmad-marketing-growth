# Audience Definition

<instructions>
Define audiences per platform from the ICP.
</instructions>

<process>
1. Read brief and `icp.md`. Pull the named ICP segments.
2. Per platform (LinkedIn, Meta, Google Demand Gen, YouTube Ads — pick
   the subset in the brief):
   - Translate the ICP segment into the platform's targeting language
     (e.g. LinkedIn: job titles, seniorities, industries, company
     size; Meta: detailed targeting + lookalikes; Google Demand Gen:
     custom segments + in-market; YouTube: affinity + custom intent).
   - State estimated audience size (cite source: platform's own
     estimator with the date you checked).
   - List one primary segment and one expansion/lookalike.
3. Audience exclusions: current customers (CRM list), competitors,
   irrelevant industries.
4. Write into `v{n}.md`.
</process>
