# Creative Brief

<instructions>
Brief the creative concept per audience.
</instructions>

<process>
1. Per audience, one concept brief covering:
   - Insight (the buyer truth the creative leans on).
   - Message (the one thing the viewer should remember).
   - Format requirements per platform (square / vertical / 16:9; video
     length; static vs motion; subtitle requirements).
   - 3 headline variants, 3 body variants (character counts per
     platform).
   - CTA per format.
2. Brand voice: validate every variant against `brand-voice.md`.
3. Anti-claims: name 2-3 things the creative must NOT say.
4. Write into `v{n}.md`.
</process>
