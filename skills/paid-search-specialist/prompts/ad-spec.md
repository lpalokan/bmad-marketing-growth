# Ad-Group & Ad-Variant Spec

<instructions>
Spec the ad groups and RSAs / PMax assets.
</instructions>

<process>
1. One ad group per intent cluster (not per keyword).
2. Per ad group: 3+ RSAs with 5+ headlines and 4+ descriptions each.
   Vary the angle (benefit, social proof, urgency, competitive).
3. Pin headlines only when necessary (compliance, trademark). State why.
4. Sitelinks, callouts, structured snippets — list the assets.
5. For PMax: separate creative asset spec (image, video, headline,
   description) and audience signals.
6. Verify every variant against `brand-voice.md` don't list.
7. Write into `v{n}.md`.
</process>
