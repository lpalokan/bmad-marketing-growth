# Keyword Tiering & Negatives

<instructions>
Build the keyword structure and the starter negatives list.
</instructions>

<process>
1. Read brief, `icp.md`, `positioning.md`.
2. Four tiers (separate ad groups):
   - Brand (our terms, exact + phrase).
   - Category (problem-space terms).
   - Competitor (named alternatives from `positioning.md`).
   - Long-tail (jobs-to-be-done phrasing).
3. For each tier: 10–20 starter keywords, match type per keyword,
   estimated monthly volume if data exists (cite source; do not
   invent).
4. Starter negatives: at least 30 terms covering job-seekers,
   tutorials, free, students, irrelevant industries.
5. Write into `v{n}.md`.
</process>
