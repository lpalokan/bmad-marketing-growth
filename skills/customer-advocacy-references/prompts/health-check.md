# Reference Health Check

<instructions>
Run a quarterly health check across the reference roster.
</instructions>

<process>
1. Per advocate: request count this quarter, last contact date,
   relationship status (CSM-reported), NPS movement.
2. Flag: over-requested (cap exceeded), under-utilized, status
   declined (renewal risk, NPS drop), retire.
3. Refresh: identify 10-20 new candidates from the latest cohort
   that hit NPS ≥ 9 + tenure ≥ 12 months.
4. Output: refreshed roster + action list.
5. Write into `v{n}.md`.
</process>
