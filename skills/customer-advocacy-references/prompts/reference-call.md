# Reference-Call Program

<instructions>
Run the reference-call program: matching, requests, hygiene.
</instructions>

<process>
1. Matching: when a prospect requests a reference, match by industry
   + size + use-case fit + advocate availability.
2. Request flow: AE → Customer Marketing → CSM → advocate. Never
   direct.
3. Hygiene: cap requests per advocate (default: 1 per quarter per
   advocate). Burnout = revoked status.
4. Post-call survey: 3 questions to the advocate; 3 questions to the
   prospect.
5. Tracking: deals attributed to a reference call.
6. Write into `v{n}.md`.
</process>
